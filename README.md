# Dotfiles

## Setup OS X

```bash
git clone git@github.com:RobertBrewitz/dotfiles.git
cd dotfiles
sh osx.sh
```

## Setup ChromeOS

### Prequisits

Activate developer mode on your Chromebook.

### Setup

```bash
git clone git@github.com:RobertBrewitz/dotfiles.git
cd dotfiles
sh chronos.sh
```
